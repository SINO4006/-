## 使用步骤
1、安装脚本，在百度网盘就可以看到红色按钮【简易下载助手】
　　如果看不到按钮，很有可能跟其它脚本冲突了，你可以尝试停用其它脚本
2、勾选1个文件，点击红色按钮【简易下载助手】，弹出下载窗口，然后在窗口上点击按钮【点击获取直链地址】
3、待获取直链地址成功后，你可以选择2种下载方式：
1）复制直链地址：复制后就可以到IDM中新建下载任务了
2）发送至Aria2：点击后直接发送至Aria2中进行下载
